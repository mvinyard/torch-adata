============
Installation
============

PYPI distribution
-----------------

To install the PYPI distribution:

.. code-block:: shell

   pip install neural-diffeqs
   

Developer version
-----------------

Alternatively, install the developer version:

.. code-block:: shell

    git clone https://github.com/mvinyard/neural-diffeqs.git; cd neural-diffeqs;
    pip install -e .
